Revolution Code Blue WordPress Theme
http://www.revolutiontwo.com/themes/code-blue

INSTALL: 
1. Upload the theme folder via FTP to your wp-content/themes/ directory.
2. Go to Admin WordPress and select Design.
3. Select Revolution Code Blue WordPress theme.
